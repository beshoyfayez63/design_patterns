- Arrow with empty triangle heads indicate inheritance.

- Arrow with empty triangle heads and dashed lines indicate that class implement an interface.

- Simple Arrow indicate that one class depends on the other.
- Simple Arrow with dashed line indicate that one class depends on the other in weak mode.

- Composition Represent by: Line with a filled diamond (container) and an arrow at the end pointing toward (the component).
- Aggregation: Represent like composition but line with an empty diamond.


- Name of abstract classes and methods are in italics.
